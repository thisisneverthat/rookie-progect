깃으로 작성하는 모든 파일에 위의 내용이 추가됨. git은 여러명이 협업하며 작업 할 수 있는 공간 이기에,
누가 작성, 수정하였는지 등등 확인이 필요하기 때문

-------------------------------

$ mkdir 디렉토리 명 : 디렉토리 생성
$ rm -r 디렉토리 명 : 디렉토리와 디렉터리 내의 파일 삭제
$ rm -rf 디렉토리 명 : 따로 의견을 묻지 않고 바로 삭제

$ pwd : 현재 디렉토리 경로 확인
$ ls : 현재 디렉토리의 디렉토리 명과 디렉토리 내의 파일명 확인
$ ls -l : long format 형식으로 정보 확인
$ ls -a : 숨김 파일 포함 모든 파일 확인(.으로 시작하는 파일은 숨김 파일)
ex) $ ls -la : 롱 포맷 형식으로 숨김 파일을 포함한 모든 파일의 정보 확인

-----디렉토리 이동 명령어-----

$ cd 디렉토리 명 : 하위 디렉토리로 이동
$ cd .. : 상위 디렉토리로 이동
$ cd . : 현재 디렉토리로 이동
$ cd ~ : 유저의 홈 디렉토리로 이동
